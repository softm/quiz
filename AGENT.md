# AGENT.md — Quiz HTML (PDF Viewer + Answer Sheet)

이 문서는 Codex(로컬 에이전트)에게 “현재 폴더의 퀴즈 웹앱”을 유지·확장하기 위한 작업 지시 프롬프트입니다.

## 목표
- `quiz.html` 하나로 다음을 제공한다.
  - 왼쪽: 시험 **문제 PDF** 뷰어(iframe)
  - 오른쪽: 1~100번 **답안지** (1~4 선택)
  - **다시풀기(리셋)** 버튼
  - **자동채점** 옵션(선택 즉시 정답/오답 표시)
  - **전체 채점** 버튼
  - **오답/미응답 필터** + 첫 항목 점프
  - **로컬 저장(localStorage)**: 새로고침/재접속해도 답안 유지
  - **섹션별 점수**: 1~25 / 26~50 / 51~75 / 76~100
  - **오답노트 자동 생성 + 복사 버튼**
  - **오답만 다시풀기(2회전 모드)**: 오답 문항만 표시, 나머지 잠금
    - 2회전 통계(대상 문항/2회전 정답/2회전 미응답)
    - 2회전 모드에서 **오답 문항만 초기화** 버튼
  - **키보드 단축키**
    - `1~4`: 현재 포커스 문항 답 선택
    - `←/→`: 현재 보이는 목록 기준 문항 이동
    - `G`: 전체 채점 + 오답노트 생성
    - `R`: 전체 리셋
    - `J`: 현재 필터(또는 2회전) 기준 첫 문항으로 점프
    - `Esc`: 2회전 모드 종료

## 파일 구성(로컬)
- `quiz.html`  (앱 본체)
- `2025년 제22회 농산물품질관리사 1차 시험문제지.pdf`  (문제)
- `최종정답.pdf` (정답표 참고용)

> PDF 파일명이 다르면 `quiz.html` 상단의 `QUESTION_PDF_URL`, `ANSWER_PDF_URL`만 수정.

## 실행 방법(권장)
브라우저에서 `file://`로 열면 PDF iframe이 막히는 경우가 있어, 로컬 서버 실행을 권장:

```bash
# macOS / Linux / Windows(파워셸도 유사)
python -m http.server 8000
# 이후 브라우저에서
# http://localhost:8000/quiz.html
```

## 정답키(ANSWER_KEY) 관리
- `quiz.html`의 `ANSWER_KEY`는 1-based 배열이며, 인덱스 1~100을 사용한다.
- 기본 UI는 1~4 선택지만 제공한다.
- 만약 실제 시험지에 5번 선지가 존재한다면(예: 일부 과목/유형) UI를 1~5로 확장해야 한다.
  - 확장 시: 선택지 생성 루프 `for(let c=1; c<=4; c++)`를 `<=5`로 변경하고,
    스타일 grid 컬럼도 5칸으로 조정(또는 자동).

## 유지보수/확장 시 규칙
- 외부 라이브러리 없이 순수 HTML/CSS/JS 유지
- 데이터 저장은 `localStorage`만 사용(서버 의존 X)
- 기능 추가 시에도 “단일 파일(quiz.html)” 원칙 유지
- UX: 버튼/토글은 상단 또는 패널 내부에 유지, 작은 화면에서도 동작해야 함
- 성능: 100문항 렌더링은 유지하되, DOM 반복을 최소화(필요 시 documentFragment)

## 향후 개선 아이디어(선택)
- 과목별 접기/펼치기(섹션 UI)
- 오답만 재시험 2회전 → 3회전(스페이스드 반복)
- 오답노트에 메모(사용자 입력) 저장 기능
- “실전 모드” (정답표시/자동채점 강제 OFF)
